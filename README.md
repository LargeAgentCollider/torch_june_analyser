# Torch JUNE Analyser
Implementation of the JUNE model using Graph Neural Networks in PyTorch

# Setup 

Install (torch_june)[https://github.com/LargeAgentCollider/torch_june] first.

Then

```bash
pip install -r requirements.txt
pip install -e .
```
